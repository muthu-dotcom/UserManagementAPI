# User Management API

This project is developed using ASP.NET Core Web API.

## Features
- CRUD operations (Create, Read, Update, Delete)
- Middleware:
  - Logging
  - Error Handling
  - Authentication
- Swagger Integration for API testing

## Technologies Used
- ASP.NET Core
- C#
- Swagger (OpenAPI)

## How to Run
```bash
dotnet run
API Endpoints

GET /api/users
GET /api/users/{id}
POST /api/users
PUT /api/users/{id}
DELETE /api/users/{id}
